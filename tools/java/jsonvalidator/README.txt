jsonvalidator
=============
This tool can be used to validate a JSON data file, also optionally against a passed JSON schema file, e.g.
jar -jar jsonvalidator.jar data.json schema.json

To run the JUnit tests
======================
ant test

To package as a single jar file
===============================
ant package

To see the usage
================
jar -jar jsonvalidator.jar
